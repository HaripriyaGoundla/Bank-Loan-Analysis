# 📊 Bank Loan Analysis Dashboard

## 📌 Project Overview

This project analyzes bank loan data using **MySQL, SQL, and Power BI**. It provides insights into customer loans, payment trends, loan types, and loan status through an interactive dashboard.

---

## 🛠️ Tools Used

- MySQL
- SQL
- Power BI
- Power Query
- DAX

---

## 📂 Dataset

The project contains:

- 500 Customers
- 1000 Loans
- 2500 Payment Records

---

## 📈 Dashboard Features

- Total Customers KPI
- Total Loans KPI
- Total Loan Amount KPI
- Average Interest Rate KPI
- Total Payments KPI
- Loan Status Distribution
- Loan Type Distribution
- Loan Amount by State
- Monthly Loan Trend
- Interactive Slicers
